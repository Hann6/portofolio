# Hans Thatian - Portfolio

Simple personal portfolio site for Hans Thatian (Backend Engineer).

## Files
- `index.html` - Main page
- `styles.css` - Styles
- `script.js` - Scroll reveal and nav animation
- `CV Hans Thatian.pdf` - Resume download

## Run locally
Open `index.html` in a browser.

## Deploy (GitHub Pages)
1. Push this repo to GitHub.
2. Go to `Settings` -> `Pages`.
3. Set `Source` to `Deploy from a branch`, choose `main` and `/ (root)`.
4. Save. Your site will be live at `https://<username>.github.io/<repo>`.
